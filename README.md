# [onequote-react](https://github.com/notosleung/onequote-react)

The `react` version of [onequote](https://github.com/notosleung/onequote)

## Dev

```bash
pnpm run dev
```

## Build

```bash
pnpm run build
```

## Preview

```bash
pnpm run build
pnpm run preview
```

## Husky init

```bash
git init # if the project already has .git, don't run this
pnpm run prepare
```

## Commitlint hint

- feat:     新功能
- fix:      修复 bug
- docs:     文档修改
- style:    代码格式修改（不影响逻辑）
- refactor: 代码重构
- test:     测试相关
- chore:    构建/工具/依赖更新

> [!TIP]
> your first commit should be with `--no-verify`:
